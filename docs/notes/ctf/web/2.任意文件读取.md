## PHP协议
> 例题：BUUCTF N1BOOK \[第一章 web入门\]afr_1

### php://filter
```url
http://example.com/php://filter/read=convert.base64-encode/resource=<file_name>
```
> 以 `Base64` 编码输出文件内容，避免 `php` 文件被服务器执行过再输出
